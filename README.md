# ThoughtNest Frontend

ThoughtNest is a modern, responsive blog platform application built with React, designed to provide a seamless blogging experience.

## Live Demo
[https://thought-nest-frontend.vercel.app/](https://thought-nest-frontend.vercel.app/)

## Key Features
- **User Authentication**: Secure Signup and Login functionality.
- **Blog Management**: Create, edit, and view blog posts.
- **Profile Management**: Manage user profiles and view personal blog collections.
- **Rich Text Editing**: Integrated rich text editor for creating content.
- **Interactive UI**: Powered by Tailwind CSS for a modern, responsive design.

## Tech Stack
- **Framework**: React 19
- **Build Tool**: Vite
- **Styling**: Tailwind CSS 4
- **Routing**: React Router 7
- **Networking**: Axios (with interceptors)
- **Utilities**:
  - `react-quill-new` for rich text editing
  - `react-hot-toast` for notifications
  - `react-select` for improved inputs
  - `js-cookie` for managing session cookies

## Getting Started
1. **Clone the repository**:
   ```bash
   git clone <repository-url>
   cd ThoughtNest
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
 
  Running Locally

   1. Run the Frontend:

   1    cd frontend
   2    npm install
   3    npm run dev

   2. Run the Backend:

   1    cd backend
   2    npm install
   3    npm run server
